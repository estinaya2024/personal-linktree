# Aya's Artistic Link-in-Bio 🌿✨

A custom-designed, stable, and "cute" personal hub built with React and TypeScript. This project features a hand-placed nature aesthetic with a curated Red, Green (#8DA750), and Beige color palette.

## ✨ Features

- **Artistic Decorations**: Unique, hand-positioned ornaments (bee, stars, bunny, headphones, tomato) with background-removed transparency.
- **Mobile Optimized**: Responsive design ensures the layout and "cute" ornaments fit perfectly on phone screens.
- **Official Social Hub**: Direct links to Instagram, Facebook, LinkedIn, GitHub, email, and Discord.
- **Personalized UI**: Charming speech bubbles for Discord and Email with a custom theme.
- **Premium Stability**: Zero animations for a high-quality, non-distracting visual experience.

## 🚀 Getting Started

### Prerequisites
- Node.js installed on your system.

### Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the development server with `npm run dev`.

### Hosting
To prepare for deployment, run:
```bash
npm run build
```
This generates a `dist/` folder ready for any static hosting provider.
